Welcome to Billing Engine Demo

A simple Flask application demonstrating billing engine concepts

Available API Endpoints

  • POST /api/customers - Create a new customer
  • GET /api/customers/<id> - Get customer details
  • POST /api/customers/<id>/payment_methods - Add payment method
  • POST /api/payments - Process payment
  • POST /api/subscriptions - Create subscription
  • POST /api/subscriptions/<id>/cancel - Cancel subscription
  • GET /api/invoices/<id> - Get invoice details

Admin Dashboard

Access the admin dashboard at: /dashboard?role=admin
